##OscilloscopeUI

Graphical user interface for AVR-based oscilloscope in Java.

### Features

* **Real-time plotting**

* **Data export**

* **Zero-level calibration**

* **Multiple channels support** *(in progress)*

### Libraries

* **jSSC-0.9.0**
https://code.google.com/p/java-simple-serial-connector/

* **JChart2D**
http://jchart2d.sourceforge.net/
